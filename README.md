# Dominos-pizza-analysis
### Power BI | SQL Server | Excel

## Project Overview
This project analyzes Domino’s Pizza sales data to uncover insights that support data‑driven decision‑making. Using SQL Server for data processing, Excel for initial exploration, and Power BI for dashboard creation, the project delivers a complete business intelligence solution.

The goal is to understand sales performance, customer behavior, product trends, and operational efficiency.

---

## Business Requirements

Domino’s management requested a set of KPIs and visual dashboards to better understand:

- Revenue performance  
- Customer ordering patterns  
- Product category trends  
- Best‑selling and underperforming pizzas  
- Sales distribution across time, size, and category  

These insights help guide marketing, inventory planning, and strategic decisions.

---

## Key Performance Indicators (KPIs)

- **Total Revenue** – Sum of all pizza order amounts  
- **Average Order Value (AOV)** – Total revenue ÷ total number of orders  
- **Total Pizzas Sold** – Sum of all pizza quantities  
- **Total Orders** – Count of all orders placed  
- **Average Pizzas Per Order** – Total pizzas sold ÷ total orders
